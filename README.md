# AI Architect Portfolio

A minimalist and elegant personal website for an AI Architecture Engineer. Designed with simplicity, clean aesthetics, and professional presentation in mind.

## Features

- **Responsive Design**: Fully responsive across all device sizes
- **Modern UI**: Clean, minimalist interface with subtle animations
- **Interactive Elements**: Smooth scrolling, form validation, hover effects
- **Performance Optimized**: Efficient CSS and JavaScript
- **Accessibility**: Keyboard navigation support and semantic HTML
- **Professional Content**: Tailored for AI/ML architecture professionals

## How to Use

### Local Development

Simply open `index.html` in your browser:

```bash
cd personal-website
# Open in default browser
start index.html  # Windows
open index.html   # macOS
xdg-open index.html  # Linux
```

### Using a Local Server (Recommended)

For best results, use a local HTTP server:

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js with http-server
npx http-server
```

Then visit `http://localhost:8000` in your browser.

### Customization

1. **Content**: Update text in `index.html`
2. **Styling**: Modify colors and styles in `css/style.css`
3. **Interactions**: Adjust JavaScript in `js/main.js`
4. **Images**: Add your images to the `images/` folder
